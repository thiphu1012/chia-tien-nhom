# brainstorm/

Scratch space for **visualization & design HTML** — self-contained pages meant to be
opened in a browser to read/think through a design, not shipped to users.

**Convention:** any HTML created as a visualization (design write-ups, flowcharts,
diagrams, dashboards for reasoning) goes in this folder — not in `public/` (which is the
deployable Mini App) and not in the repo root.

Each file should be a single self-contained `.html` (inline CSS/JS, no external assets)
so it opens offline with a double-click.
